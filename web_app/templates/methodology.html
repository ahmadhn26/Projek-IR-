Overview

Penelitian ini menganalisis sentimen masyarakat terhadap rencana kenaikan tarif Transjakarta menggunakan data dari platform media sosial TikTok dan YouTube. Analisis dilakukan menggunakan tiga algoritma machine learning: Naive Bayes, SVM (Support Vector Machine), dan Random Forest.

Metrics & Comparison

1. Preprocessing

  • Text Cleaning: Hapus emoji, URL, mention, angka
  • Case Folding: Konversi ke lowercase
  • Normalisasi Slang: Kamus alay khusus Transjakarta
  • Stopword Removal: Dengan mempertahankan kata negasi
  • Stemming: Menggunakan Sastrawi

2. Feature Extraction

  • Metode: TF-IDF Vectorizer
  • N-gram: (1, 2) - unigram & bigram
  • Max Features: 3000
  • Min DF: 2 (kata muncul minimal 2x)
  • Max DF: 0.8 (maksimal di 80% dokumen)

3. Data Balancing

  • Metode: SMOTE (Synthetic Minority Over-sampling)
  • Aplikasi: Hanya pada training set
  • Train-Test Split: 80:20 dengan stratified sampling
  • Tujuan: Mengatasi imbalanced data

4. Hyperparameter Tuning

  • Metode: GridSearchCV
  • Cross-Validation: 5-fold CV
  • Scoring: F1 Macro Score
  • Optimisasi: Parameter terbaik untuk setiap model

Model Machine Learning

Naive Bayes

Model probabilistik berbasis Teorema Bayes. Efisien untuk text classification.

Tuned Parameters:
alpha: [0.1, 0.5, 1.0, 2.0, 5.0]

SVM

Support Vector Machine dengan kernel linear dan RBF. Optimal untuk high-dimensional data.

Tuned Parameters:
C: [0.1, 1, 10, 100]
kernel: ['linear', 'rbf']

Random Forest

Ensemble learning dengan multiple decision trees. Robust terhadap overfitting.

Tuned Parameters:
n_estimators: [50, 100, 200]
max_depth: [10, 20, None]

Metrik Evaluasi

Primary Metrics

  • F1 Macro Score: Metrik utama untuk binary classification
  • Accuracy: Persentase prediksi yang benar
  • ROC-AUC: Kemampuan model membedakan kelas

Additional Metrics

  • Precision: Akurasi prediksi positif
  • Recall: Kemampuan mendeteksi kelas positif
  • Confusion Matrix: Analisis error per kelas
